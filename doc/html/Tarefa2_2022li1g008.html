:: [Obstaculo]

-> Bool

A função retorna o valor lógico True se não existirem mais do que 5 troncos consecutivos nem mais do que 3 carros consecutivos.

A função listaObstaculos verifica se não existem mais do que 5 troncos consecutivos nem mais do que 3 carros consecutivos.

tamanhoObstaculos #

Arguments

:: [[Obstaculo]]

A função recebe uma lista de listas de obstáculos.

-> Bool

Caso o primeiro elemento de uma lista seja Tronco ou Carro e o comprimento da lista seja maior que 5 ou que 3, respetivamente, o resultado será False.

A função tamanhoObstaculos verifica se não existem listas de comprimento superior a 5 e com um Tronco como primeiro elemento dessa lista, e se não existem listas de comprimento superior a 3 e com um Carro como primeiro elemento dessa lista.
